SECTION 5 — GOOGLE SHEET STRUCTURE
═══════════════════════════════════════════

ONE MASTER FILE
FOUR TABS INSIDE

ALL TABS SAME COLUMNS:
Col A → Order ID
Col B → Customer Name
Col C → Phone (91XXXXXXXXXX)
Col D → Status
Col E → Tracking Link
Col F → Message Sent (YES/NO/FAILED)
Col G → Last Updated

TAB 1 → Anjani
TAB 2 → DTDC
TAB 3 → Delhivery
TAB 4 → Others

CONTACTS SHEET (for campaigns):
Col A → Customer Name
Col B → Phone (91XXXXXXXXXX)
Col C → Last Order Date
Col D → City
Col E → Tags
Col F → Opted In (YES/NO)

CAMPAIGNS SHEET (log):
Col A → Campaign ID
Col B → Campaign Name
Col C → Template Used
Col D → Audience Count
Col E → Sent Count
Col F → Failed Count
Col G → Status
Col H → Created At
Col I → Sent At
